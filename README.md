# 🦁 Lions Portfolio — Leonardo G. Mendoza

Site pessoal de portfólio com Lions Agent (IA powered by Gemini).

**URL:** [lions.sandlj.com.br](https://lions.sandlj.com.br)

## 🚀 Como rodar localmente

Basta abrir o `index.html` no navegador! Não precisa de build.

## 🤖 Lions Agent

O chat usa a API Gemini. Para ativar:
1. Acesse [aistudio.google.com/apikey](https://aistudio.google.com/apikey) e gere uma chave **grátis**
2. Abra o site e clique em 🦁
3. Cole sua API Key no campo e clique em **Salvar**
4. A chave fica salva no navegador (localStorage)
